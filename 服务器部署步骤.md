## 第二步：导入数据库

```bash
mysql -u root -p student_management < database-mariadb.sql
```

输入MySQL密码

## 第三步：配置后端环境

```bash
cd server
vi .env
```

按 `i` 进入编辑模式，修改 `DB_PASSWORD` 为你的MySQL密码，按 `ESC`，输入 `:wq` 保存

## 第四步：安装后端依赖并启动

```bash
npm install
npm install -g pm2
pm2 start index.js --name student-api
pm2 save
pm2 startup
```

## 第五步：验证后端

```bash
pm2 status
curl http://localhost:3001/api
```

## 第六步：获取服务器IP

```bash
curl ifconfig.me
```

记下这个IP地址

## 第七步：修改前端API地址

```bash
cd /var/www/student-management-system
vi src/utils/request.js
```

找到 `baseURL` 这一行，改成：
```javascript
baseURL: 'http://你的服务器IP:3001/api',
```

## 第八步：构建前端

```bash
npm install
npm run build
```

## 第九步：安装Nginx

```bash
yum install -y nginx
systemctl start nginx
systemctl enable nginx
```

## 第十步：配置Nginx

```bash
vi /etc/nginx/conf.d/student.conf
```

粘贴以下内容（记得替换IP）：

```nginx
server {
    listen 80;
    server_name 你的服务器IP;

    location / {
        root /var/www/student-management-system/dist;
        try_files $uri $uri/ /index.html;
    }

    location /api {
        proxy_pass http://localhost:3001;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

重启Nginx：
```bash
nginx -t
systemctl restart nginx
```

## 第十一步：开放防火墙

```bash
firewall-cmd --permanent --add-port=80/tcp
firewall-cmd --permanent --add-port=3001/tcp
firewall-cmd --reload
```

## 第十二步：配置阿里云安全组

1. 登录阿里云控制台
2. ECS → 网络与安全 → 安全组
3. 配置规则 → 添加安全组规则
4. 开放端口：80、3001

## 完成！
