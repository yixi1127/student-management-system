# 头像设置指南

## 功能说明

个人中心现在支持自定义头像，通过输入图片URL地址来设置头像。

## 使用方法

### 方法1: 使用免费图床服务（推荐）

1. **ImgBB** (https://imgbb.com/)
   - 免费、无需注册
   - 上传图片后复制"Direct link"
   - 粘贴到头像URL输入框

2. **SM.MS** (https://sm.ms/)
   - 免费、支持中文
   - 上传图片后复制图片链接
   - 粘贴到头像URL输入框

### 方法2: 使用Gravatar（基于邮箱）

1. 访问 https://cn.gravatar.com/
2. 使用你的邮箱注册并上传头像
3. 使用以下格式的URL:
   ```
   https://www.gravatar.com/avatar/你的邮箱MD5值?s=200
   ```

### 方法3: 使用其他图片链接

任何可以公开访问的图片URL都可以使用，例如：
- 你的个人网站上的图片
- 其他图床服务的图片
- CDN上的图片

## 操作步骤

1. 进入"个人中心"页面
2. 找到"头像"设置项
3. 在输入框中粘贴图片URL
4. 左侧会实时预览头像效果
5. 点击"保存修改"按钮
6. 刷新页面，右上角头像也会更新

## 推荐图片规格

- 尺寸: 200x200 像素或更大
- 格式: JPG, PNG, GIF
- 大小: 建议小于1MB

## 示例URL

```
https://i.imgur.com/example.jpg
https://i.ibb.co/example/avatar.png
https://www.gravatar.com/avatar/205e460b479e2e5b48aec07710c08d50?s=200
```

## 注意事项

1. **URL必须是公开可访问的**
   - 不能使用本地文件路径
   - 不能使用需要登录才能访问的图片

2. **建议使用HTTPS链接**
   - 更安全
   - 避免浏览器安全警告

3. **图片加载速度**
   - 选择稳定的图床服务
   - 避免使用过大的图片

4. **隐私保护**
   - 不要上传包含敏感信息的图片
   - 使用正规的图床服务

## 常见问题

### Q: 为什么我的头像不显示？
A: 检查以下几点：
- URL是否正确
- 图片是否可以公开访问
- 图片格式是否支持（JPG/PNG/GIF）
- 网络连接是否正常

### Q: 可以上传本地图片吗？
A: 需要先将本地图片上传到图床服务，然后使用图床提供的URL。

### Q: 头像多久会更新？
A: 保存后立即生效，刷新页面即可看到新头像。

### Q: 可以使用动图吗？
A: 可以，支持GIF格式的动态图片。

## 技术说明

- 头像URL存储在数据库的 `users.avatar_url` 字段
- 前端使用 Element Plus 的 `el-avatar` 组件显示
- 如果没有设置头像，会显示默认头像

## 安全提示

1. 只使用可信的图床服务
2. 不要点击不明来源的图片链接
3. 定期检查头像是否正常显示
4. 如果图床服务关闭，及时更换头像URL
