# 如何修改学生姓名

## 方法1：通过界面修改（推荐）

1. 登录系统
2. 进入"学生管理"页面
3. 找到要修改的学生，点击"编辑"按钮
4. 在"真实姓名"字段输入中文姓名（如：张三）
5. 点击"确定"保存

**优点：**
- 简单直接
- 不会有编码问题
- 可以同时修改其他信息

## 方法2：使用MySQL Workbench

1. 打开MySQL Workbench
2. 连接到数据库
3. 执行以下SQL：

```sql
USE student_management;

UPDATE students SET real_name = '张三' WHERE student_no = '2024001';
UPDATE students SET real_name = '李四' WHERE student_no = '2024002';
UPDATE students SET real_name = '王五' WHERE student_no = '2024003';
UPDATE students SET real_name = '赵六' WHERE student_no = '2024004';
```

4. 点击执行（闪电图标）

## 注意事项

- 修改后需要刷新页面才能看到效果
- 姓名会同步显示在所有相关页面（考勤、成绩、选课等）
- 建议使用真实姓名，方便管理
