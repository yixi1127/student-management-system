# 学生管理系统 - 项目复现教程

## 项目概述

这是一个基于 Vue3 + Node.js + MySQL 的学生管理系统，包含以下功能：
- 用户登录注册（管理员）
- 学生信息管理（增删改查）
- 课程管理
- 选课管理
- 成绩管理
- 考勤管理
- 通知管理
- 数据统计（图表展示）

## 技术栈

### 前端
- Vue 3 - 渐进式JavaScript框架
- Vite - 新一代前端构建工具
- Element Plus - Vue 3 UI组件库
- Vue Router - 路由管理
- Pinia - 状态管理
- Axios - HTTP请求库
- ECharts - 数据可视化图表库

### 后端
- Node.js - JavaScript运行环境
- Express - Web应用框架
- MySQL - 关系型数据库
- mysql2 - MySQL数据库驱动
- jsonwebtoken - JWT身份验证
- bcryptjs - 密码加密
- cors - 跨域资源共享

## 项目结构

```
student-management-system/
├── server/                 # 后端代码
│   ├── config/            # 配置文件
│   ├── middleware/        # 中间件
│   ├── routes/            # 路由
│   ├── index.js           # 入口文件
│   └── package.json       # 后端依赖
├── src/                   # 前端代码
│   ├── assets/            # 静态资源
│   ├── components/        # 组件
│   ├── router/            # 路由配置
│   ├── stores/            # 状态管理
│   ├── utils/             # 工具函数
│   ├── views/             # 页面
│   ├── App.vue            # 根组件
│   └── main.js            # 入口文件
├── database-mysql.sql     # 数据库脚本
├── package.json           # 前端依赖
└── vite.config.js         # Vite配置
```

1. **环境准备** - 安装必要的开发工具
2. **数据库设计** - 创建数据库和表结构
