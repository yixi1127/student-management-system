# 第一步：环境准备

## 1. 安装Node.js

### 下载安装
1. 访问 https://nodejs.org/
2. 下载LTS版本（推荐18.x或20.x）
3. 运行安装程序，一路Next
4. 验证安装：
```bash
node -v
npm -v
```

## 2. 安装MySQL

### 下载安装
1. 访问 https://dev.mysql.com/downloads/mysql/
2. 下载MySQL Community Server
3. 安装时记住设置的root密码（例如：Gpnu115214@）
4. 验证安装：
```bash
mysql --version
```

### 启动MySQL服务
Windows:
```bash
net start mysql
```

## 3. 安装开发工具

### 推荐使用VS Code
1. 访问 https://code.visualstudio.com/
2. 下载并安装
3. 安装推荐插件：
   - Vue Language Features (Volar)
   - ESLint
   - Prettier

## 4. 创建项目文件夹

```bash
mkdir student-management-system
cd student-management-system
```

## 5. 验证环境

确保以下命令都能正常运行：
```bash
node -v          # 应显示版本号，如 v20.x.x
npm -v           # 应显示版本号，如 10.x.x
mysql --version  # 应显示版本号
```
