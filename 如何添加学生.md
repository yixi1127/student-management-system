# 如何添加学生信息

## 功能说明
系统已经具备完整的学生信息管理功能,包括添加、编辑、删除和查看学生信息。

## 添加学生步骤

### 1. 进入学生管理页面
- 登录系统后,点击左侧菜单的"学生管理"

### 2. 点击"添加学生"按钮
- 在页面顶部找到"添加学生"按钮(蓝色按钮)
- 点击后会弹出添加学生的对话框

### 3. 填写学生信息
必填字段:
- **学号**: 唯一标识,不能重复(如: 2024001)
- **姓名**: 学生真实姓名
- **性别**: 选择男或女
- **年级**: 选择年级(2021-2024级)
- **班级**: 输入班级(如: 1班、计算机1班)
- **联系电话**: 11位手机号码
- **邮箱**: 有效的邮箱地址(用于登录系统)

可选字段:
- **家庭地址**: 学生家庭住址
- **紧急联系人**: 紧急联系人电话

### 4. 提交保存
- 填写完成后点击"确定"按钮
- 系统会自动创建学生账号,默认密码为: `123456`
- 学生可以使用邮箱和密码登录系统

## 重要说明

### 自动创建账号
添加学生时,系统会自动:
1. 在`users`表中创建一个用户账号
2. 角色设置为`student`(学生)
3. 默认密码为`123456`
4. 在`students`表中创建学生详细信息
5. 通过`user_id`关联用户和学生信息

### 学号规则
- 学号必须唯一,不能重复
- 建议格式: 年级+序号(如: 2024001, 2024002)
- 系统会自动检查学号是否已存在

### 邮箱规则
- 必须是有效的邮箱格式
- 用于学生登录系统
- 建议使用学校邮箱或个人常用邮箱

### 手机号规则
- 必须是11位数字
- 以1开头,第二位为3-9
- 格式: 1[3-9]xxxxxxxxx

## 其他功能

### 编辑学生
- 点击表格中的"编辑"按钮
- 修改学生信息后保存

### 删除学生
- 点击表格中的"删除"按钮
- 确认后会同时删除用户账号和学生信息

### 查看学生详情
- 点击表格中的"查看"按钮
- 弹窗显示学生完整信息

### 搜索学生
使用顶部搜索栏可以按以下条件搜索:
- 学号(模糊搜索)
- 姓名(模糊搜索)
- 年级(精确匹配)
- 班级(模糊搜索)

## 数据库直接添加(高级)

如果需要批量添加大量学生,可以直接在MySQL中执行SQL:

```sql
-- 1. 先添加用户
INSERT INTO users (email, password, real_name, phone, role, status) 
VALUES ('student5@example.com', '123456', '赵六', '13800138005', 'student', 1);

-- 2. 获取刚才插入的用户ID(假设是5)
-- 3. 添加学生信息
INSERT INTO students (user_id, student_no, grade, class, gender, address, emergency_contact) 
VALUES (5, '2024005', '2024', '计算机1班', '男', '广州市天河区xxx', '13900139005');
```

## 常见问题

### Q: 为什么添加失败?
A: 可能原因:
1. 学号已存在(检查是否重复)
2. 邮箱格式不正确
3. 手机号格式不正确
4. 必填字段未填写

### Q: 学生如何登录?
A: 学生使用添加时填写的邮箱和默认密码`123456`登录系统

### Q: 如何修改学生密码?
A: 学生登录后可以在"个人中心"修改密码,或管理员直接在数据库中修改

### Q: 删除学生会删除相关数据吗?
A: 是的,删除学生会同时删除:
- 用户账号
- 学生基本信息
- 相关的成绩记录(如果有外键约束)
- 相关的考勤记录(如果有外键约束)

## 测试建议

建议添加不同类型的学生进行测试:
1. 不同年级的学生(2021-2024)
2. 不同班级的学生
3. 男女学生各若干
4. 确保学号、邮箱不重复

这样可以更好地测试:
- 成绩录入功能
- 考勤统计功能
